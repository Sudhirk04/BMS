###########################
Manual MinGW64 Installation
###########################

#. Download MinGW-W64 version ``x86_64-posix-seh`` from
   `sourceforge.net <https://sourceforge.net/projects/mingw-w64/files/mingw-w64/>`_
#. Extract the archive
#. Copy the extracted ``mingw64`` directory to
   ``C:\mingw64\x86_64-8.1.0-release-posix-seh-rt_v6-rev0``.
