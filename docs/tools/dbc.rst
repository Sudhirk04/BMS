DBC File
========

..
    Comments:


.. spelling::
    sym

These files describe the CAN interface used by |foxbms|. The .dbc-file has been
created using PCAN Symbol Editor |version_pcan_se| from symbol file
|version_sym_file|.

The .dbc-file and .sym-file are located in ``tools/dbc``.
